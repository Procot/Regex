Как необходимо оформить решение задачи для отправки на сервер на проверку?

Всем привет!

С Вами снова Procot и в данной статье я хочу рассказать про оформление решения задачи.

И вот Вы решили задачу, надеюсь что успешно и Ваше решение с первого же раза пройдет все тесты. Первая проблема, которая может возникнуть на вашем этапе - это конечно же как оформить решение, чтобы оно "зашло".

В данном проекте тестировщик требует оформление решения в виде функции, входными параметрами которой будут являться вводимые данные программы. Функция должна возвращать результат работы программы.

Так как в основном задачи на данном сайте имеют только один вводимые параметр и один выходной параметр, то решение может быть оформлено по следующему шаблону:

		function(input_data) {

		    //решение задачи

		    return output_data;

		}
